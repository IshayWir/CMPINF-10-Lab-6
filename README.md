# CMPINF-10-Lab-6
## Something™ Program

## Description

This Python program is an interactive command-line application that asks the user for their name and a number. After receiving the input, the program performs a calculation on the number and displays a personalized message with the result.

## What the Program Does

1. Prompts the user to enter their name.
2. Prompts the user to enter a number.
3. Performs one of the following actions:
   - If the number is a non-negative whole number, it calculates the factorial of that number.
   - If the number is not a whole number, it calculates the square of the number.
4. Displays a personalized message that includes:
   - The user’s name
   - The number they entered
   - The result of the calculation

## Example

If the user enters:

Name: Alex  
Number: 5  

The program will calculate the factorial of 5 and output:

Hello, Alex!  
You entered 5.  
Something™ happened... the factorial is 120!

## Purpose

This program demonstrates:
- User input handling
- Conditional logic
- Loops
- Basic mathematical operations
- Formatted output in Python
